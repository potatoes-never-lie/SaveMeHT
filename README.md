# SaveMeHT

실행 전에 DB 설정(user, password, db 등) 바꿔주어야 합니다!!

## join 실행방법

```
1.터미널에 nodemon app.js 입력
2.http://localhost:3000/join 으로 접속

```
## login 실행방법

```
1.터미널에 nodemon app.js 입력
2.http://localhost:3000/login 으로 접속

```
